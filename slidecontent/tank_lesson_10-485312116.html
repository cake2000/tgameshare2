3. Health Regeneration Power

Health recovers by 240 * (1 + Regen_Points) per second

Regen_Points of 3 => 240 * (1 + 3) = 960 per second

4. Reload Power

Shell reload time = (1200 - 100 * Reload_Points) milliseconds

If Reload_Points = 0, shell reload time = 1200 ms

If Reload_Points = 2, shell reload time = 1000 ms

If Reload_Points = 4, shell reload time = 800 ms

A series of "if-else" statements that specify what actions to take under different conditions

Logical Requirements for Rule-Based Systems

Self-Consistent

The rules cannot contradict each other.

Complete

The rules have to cover all possible situations.

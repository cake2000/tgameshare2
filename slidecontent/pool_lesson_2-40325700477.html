All Shot Command Properties

Property Value Range Property Meaning
aimx any number x coordinate of the aiming point
aimy any number y coordinate of the aiming point
strength 0 to 100 how much strength to apply
spin -1 to 1 vertical spin (forward or backward spin)
hspin -30 to 30 horizontal spin (right 30 degrees to left 30 degrees)
cueballx -935 to -500 x coordinate of initial cueball placement for break shot
cuebally -435 to 435 y coordinate of initial cueball placement for break shot

Syntax Rules for Property Names

name : value

  • Any string in a pair of ' ' or " "
  • Simple format: A-Z, number, $ and _ (can't start with a number)

  • Case sensitive: "lastName" is different from "lastname"
{
  "ID#":  2051, "last name": "Adams", "score2$": 300
}
{
  ID_Number: 2051, lastName: "Adams", score2$: 300
}

Types of Property Values

  • Numbers: 1.5, -23
  • Strings: "red", 'New York'
  • Objects
{
  bodyColor: "red",
  seatCount: 2,
  wheel: {
    color:  "black",
    radius: 17
  }
}

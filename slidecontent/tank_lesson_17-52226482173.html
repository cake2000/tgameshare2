Array Methods

Tanks, Crystals, Weapons, Maze, ...

  • array.push(element1, element2, ...): adds one or more new elements to the end of the array.
  • array.includes(element): checks whether the array contains the given element.
  • array.find(test_function): returns the first element that satisfies the given test.
  • array.filter(test_function): returns a new array with all elements satisfying the given test.
  • array.slice(startIndex, endIndex): returns the elements between startIndex and endIndex as a new array

Array.Push

array.push(element1, element2, ...): adds one or more new elements at the end of the array.

Examples:

var nums = [2, 4, 6];
nums.push(8, 10);
// nums becomes [2,4,6,8,10]
var tanks = [
  {color: "blue", x: 20, y: 30},
  {color: "white", x: 120, y: 230},
];

tanks.push( { color: "red", x: 44, y: 300});
// tanks contain 3 object elements now
